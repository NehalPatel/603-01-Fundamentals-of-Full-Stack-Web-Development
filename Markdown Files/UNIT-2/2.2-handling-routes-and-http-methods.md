# Handling Routes and HTTP Methods

## Learning Objectives

- Define resource routes
- Map GET/POST/PUT/DELETE to CRUD
- Organize routes with `Router`

## Code Examples

```js
import express from 'express'
const app = express(); app.use(express.json())
const r = express.Router()
r.get('/', (req, res) => res.json([]))
r.post('/', (req, res) => res.status(201).json(req.body))
r.put('/:id', (req, res) => res.json(req.body))
r.delete('/:id', (req, res) => res.sendStatus(204))
app.use('/api/items', r)
app.listen(3000)
```

## Practical Exercises

### Exercise

- Add `/:id` GET route returning 404 when missing

### Solution

```js
r.get('/:id', (req, res) => res.sendStatus(404))
```

## Troubleshooting & Pitfalls

- Ensure route order from specific to generic
- Always return appropriate status codes
