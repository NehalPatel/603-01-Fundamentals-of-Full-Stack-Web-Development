# Define Routes: GET, POST, PUT, DELETE

## Code Examples

```js
import express from 'express'
const r = express.Router()
r.get('/', (req, res) => res.json([]))
r.post('/', (req, res) => res.status(201).json(req.body))
r.put('/:id', (req, res) => res.json(req.body))
r.delete('/:id', (req, res) => res.sendStatus(204))
```

## Troubleshooting & Pitfalls

- Use `201` for created resources
- Return `404` when item not found
